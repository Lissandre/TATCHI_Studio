# TATCHI_Studio website

Website build with Nuxt & Strapi

## Front Build Setup

Go to ./frontend

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Back Build Setup

Go to ./backend

```bash
# install dependencies
$ npm install

# serve at localhost:1337
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# other commands
$ npm run strapi
```
